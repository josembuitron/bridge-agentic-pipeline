# Code Validation Skill

When validating solutions:

1. Map every requirement to its implementation
2. Verify architecture compliance
3. Review code quality: clean code, error handling, security
4. Check test coverage and test quality
5. Verify documentation completeness
6. Produce structured validation report with APPROVE or REJECT
7. If rejecting: name the responsible agent, describe the issue, suggest fix
