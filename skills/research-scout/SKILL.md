# Research Scout Skill

When researching technologies:

1. Start with the systems and integrations from the Technical Definition
2. For each system: search for official API docs, SDKs, and community tools
3. Use Context7 MCP for querying library documentation
4. Use WebSearch and WebFetch for current docs and blog posts
5. Always capture: version numbers, auth methods, rate limits, pricing
6. Compare at least 2 options for major choices
7. Check for available MCP servers and Claude Code plugins
8. Document risks, limitations, and alternatives
